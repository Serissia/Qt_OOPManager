# 面向对象程序项目管理系统 

 SEU语言程序设计课设作业 

## 一 · 题目

该系统需创建和管理如下信息：

1. 类信息：编号、类名、基类名、功能、创建日期、作者、类成员数据集；
2. 类成员信息：成员编号、成员名称、成员类型（数据或函数）、内存字节数（数据成员）、数据类型、可访问性（公有、私有、保护）等。 

系统功能要求如下：  

1. 创建和管理类信息的对象；  

2. 创建和管理类成员信息的对象；  

3. 增加和删除类；  

4. 针对特定类增加或删除类成员信息；  

5. 基本查询功能；  

6. 数据文件读写：文件中包含所有类信息、每个类的类成员信息等数据； 

7. 基本信息显示：1）所有类的信息显示；2）特定类的类成员信息；  

8. 可选功能提升：显示类对象的占用内存信息等。
